Displays the number of ratings a plugin has received without having to hover over the stars in your site's admin panel when browsing for plugins.

== Description ==

Shows you how many ratings a plugin has received without having to hover over the stars in your site's admin panel when browsing for a new plugin.
Additional enhancements are coming soon as well!

== Installation ==

Extract the zip file and drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the plugin from the "Plugins" page.
